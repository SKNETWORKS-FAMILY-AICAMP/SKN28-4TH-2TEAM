# 와이어프레임 — 채팅 (메인)

    +----------------+--------------------------------------+-------------------+
    | 세션 레일      |  헤더: 제목 · 학과 안내      [관리자][테마] |  넙죽이 패널       |
    | [+ 새 대화]    |--------------------------------------|  (마스코트)        |
    | [대화 검색]    |                                      |   상태칩: 대기/검색 |
    | 둘러보기       |   사용자 말풍선 (오른쪽)             |   완료/경고        |
    |  · 학과 목록   |                                      |   한마디           |
    | AI대학         |   넙죽이 답변 (왼쪽)                 |                   |
    |  · 전공 과목   |    - 본문 (LLM 생성)                |   넙죽이의 팁      |
    |  · 교수진      |    - 출처 카드 1..n (클릭→원문)      |                   |
    | 자연과학대학    |    - 👍 👎 복사 다시생성             |                   |
    | 생명과학기술대 |                                      |                   |
    |  ...           |--------------------------------------|                   |
    | [사용자/로그아웃]| 추천칩 · [입력창...........][전송]   |                   |
    +----------------+--------------------------------------+-------------------+

- 입력 → thinking(생각중) → source(검색) → done(완료) 로 마스코트 상태 전이.
- 출처 카드: 학과·문서명·수집일 + URL(클릭 시 새 탭).
- 범위 밖/미수집 질문 → warning(경고) 답변.
