    |                |  | 작성일 · 댓글 n개              |   |  유형을 골라      |
    |                |  +--------------------------------+   |  적어주세요.       |
    |                |  |[답변 완료] 유형  제목 …        |   |                   |

    [문의 작성 모달]                      [문의 상세 모달]
    +----------------------------+        +----------------------------+

    |  게시판 기능/기타)          |        | 내용 ......................  |
    | 제목 [................]    |        | --- 운영자 답변 ----------   |
    | 내용 [................]    |        | 답변(댓글) ................  |
    | □ 비공개로 문의            |        | (관리자) 상태 ▼ 대기/처리중/ |
    | [취소]            [등록]   |        |          완료  [답변 등록]    |
    +----------------------------+        +----------------------------+

- 답변 상태: 답변 대기(wait) · 처리중(progress) · 답변 완료(done) — 목록·상세에 상태칩 표시.
- 문의 유형: 계정 및 로그인 · 데이터 오류 · 답변 품질 · 게시판 기능 · 기타.
- 비공개(is_private): 기본 비공개. 작성자 본인과 관리자만 내용 열람, 목록엔 🔒 표시.
- 작성은 로그인 필요. 비로그인 시 좌측 "내가 쓴 문의글"에 "로그인 후 확인" 안내.
- 권한: 작성자는 본인 문의 수정·삭제, 관리자는 상태 변경 + 답변(댓글) 작성.
- 답변 알림 이메일은 현재 비활성(email_on_answer=false).
- 예외: 빈 유형/제목/내용 → 인라인 오류, 로딩 스켈레톤, 실패 시 재시도 안내.
- API: `GET/POST /api/community/inquiries/`, `/inquiries/<id>/`, `/inquiries/<id>/comments/`, `/inquiries/<id>/status/`.
