# 화면설계서 — 게시판 (커뮤니티)

> 학사 정보 공유·질의를 위한 보완 Q&A 게시판. 챗봇 답변을 보완하는 사용자 간 정보 교류 공간.

| 라우트 | 접근 | 화면 구성 | 연동 API |
|---|---|---|---|
| `/board/` | 열람: 누구나 / 작성·댓글: 로그인 | 목록 · 상세 · 글쓰기 | `/api/community/posts/`·`/posts/<id>/`·`/posts/<id>/comments/` |

---

## 1. 게시판 목록

![게시판 목록 번호 표기](images/screens/annotated/board_list.png)

| 번호 | 화면 요소 | 설명 / 동작 |
|:--:|---|---|
| ① | 새 글 작성(레일) | 글쓰기 화면 진입 (비로그인 시 로그인 유도) |
| ② | 검색창 | 제목·내용·카테고리·작성자 부분일치(입력 디바운스) |
| ③ | 글쓰기 버튼 | 글 작성 화면 진입 |
| ④ | 게시글 수 | 총 건수 · "한 페이지에 5개씩 표시" |
| ⑤ | 공지 글 | `is_notice` 글은 상단 고정 + amber 뱃지 |
| ⑥ | 게시글 행 | 카테고리 뱃지 · 작성자 · 제목 · 요약 · 조회/댓글/작성일 |
| ⑦ | 페이지네이션 | 5개 단위, `« ‹ 1 2 3 › »`, 현재 페이지 active |
| ⑧ | 마스코트 패널 | 상태칩·팁 (커뮤니티 공통) |

## 2. 게시글 상세

![게시글 상세 번호 표기](images/screens/annotated/board_detail.png)

| 번호 | 화면 요소 | 설명 / 동작 |
|:--:|---|---|
| ① | 목록으로 | 목록 화면 복귀 |
| ② | 수정 / 삭제 | 작성자 본인(`canManage`)에게만 노출 |
| ③ | 카테고리·공지 뱃지 | 글 분류 및 공지/댓글차단 표시 |
| ④ | 작성자 카드 | 이니셜 아바타 · 이름 · 조회/댓글/작성일 |
| ⑤ | 본문 | 글 내용(+ 참고 링크 있을 시 새 탭 카드) |
| ⑥ | 댓글/대댓글 | 작성자명 + 내용, 본인 댓글은 수정/삭제 |
| ⑦ | 댓글 입력창 | 로그인 시 입력 / 게스트·차단 글은 비활성 |

## 3. 글쓰기 폼 · 검증

| 필드 | 타입 | 필수 | 제약 |
|---|---|:--:|---|
| 카테고리 | select | ✅ | 학사·수강·장학·졸업·연구·생활·자유 |
| 제목 | text | ✅ | 최대 160자 |
| 본문 | textarea(10행) | ✅ | — |
| 참고 링크 | text(URL) | ❌ | 최대 500자, 상세에서 새 탭 링크 |
| 공지로 표시 / 댓글 차단 | checkbox | ❌ | **관리자만** 노출 |

> 서버 검증(`_validate_post_payload`): 카테고리·제목·본문 누락 시 `400` + 한글 안내.
